focando em imagem parte de uma imagem , no centro


podemos fazer muitas coisas uteis com a propriedade object-fit das imagens

Se usarmos object-fit: cover;a imagem mantém sua proporção e preenche a dimensão dada. A imagem será cortada para caber mesmo quando ela é enorme e queremos que a exibição seja em algo pequeno:

Se usarmos object-fit: contain;a imagem mantém sua proporção, mas é redimensionada para caber na dimensão especificada porém sem esticar;

Se usarmos object-fit: fill;a imagem é redimensionada para preencher a dimensão dada. Se necessário, a imagem será esticada ou comprimida para caber :fica baparecendo pixels devido esticamento

Se usarmos object-fit: scale-down;a imagem é reduzida para a menor versão de none ou contain:
