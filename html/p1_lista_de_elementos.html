Lista no html

No html temos lista ordenadas ol e não ordenadas ul, e a aparecia e indice de ordenação desses itens são modificados pela propriedade type que tanto o Ol quanto ul possui:

ol possui os typs: 1 (numericos), algarismos romano(I,i), alfabeticos(A,a)

  1. Maçã
  2. Macarão
  3. Feijoada

ul possui os typs: disc(disco preenchido), circle(disco com meio não preenchido), square(quadrado), none (não aparece)

  • Maçã
  • Macarão
  • Feijoada
        
            <ol type="i">
                <li>Maçã</li>
                <li>Macarão</li>
                <li>Feijoada</li>
            </ol>
        
            <ul type="disc">
                <li>Maçã</li>
                <li>Macarão</li>
                <li>Feijoada</li>
            </ul>
