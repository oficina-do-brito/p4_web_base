sessão reset de css

Para que serve o resetar o css?

Bem, resetar o css serve para que nos como programadores e usuario tenhamos uma visualização unificada dos nossos softwares independente do uso de qualquer navegador.

Tudo parte da ideia de que no html por padrão Qualquer arquivo criado já vem com propriedades css padronizadas de margens e outras padrões que são diferentes em cada navegador, por exemplo o chrome tem padrão de espaçamento diferente do mozila, do opera e por ai vai , e a gente como programador quer resetar esse estilo ao 0 para todos os tipos de navegadores e daí depois de resetado queremos fazer nosso próprio estilo unificado.

Outro conceito que precisamos saber é que por padrão o tamanho de texto vem em 16px(Root é 16px) que equivale no word a um tamanho de letra 12. Só que unidade em px deixam os layout fixos(ou seja, com dificuldade de se adaptar) então nossa estratégia é utilizar rem que é um tamanho fluido no qual é multiplicado com base em um caractere M root ou pai: 

            
                * {
                    margin: 0;
                    padding: 0;
                    box-sizing: border-box;
                }
                html{
                        /* 1rem = 10px */
                        font-size: 62.5%;
                }
                body {
                    width: 100%;
                    height: 100%;
                    font-family: 'Roboto', sans-serif;
                    font-size: 1.6rem;
                    background-color: #f5f5f5;
                }
