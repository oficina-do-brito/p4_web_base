Para trabalhar com grids no css precisamos informar no elemento que ele é um display:grid;

Trabalhamos com grid por que com ele é possível trabalhar em duas dimensões ou seja, no eixo x e no y e ainda em forma de linha e coluna, o que nos facilita bastante a vida . Quando trabalhamos com grid devemos pensar em modos de trabalho, por exemplo se vamos mexer com o grid container ele vai ter as propriedades x, porém para grid itens ele já vai ter outras propriedades. Container:

  1. display:grid:
  2. grid-template-columns: auto; - Quantas colunas o grid possui.
  3. grid-template-rows: auto;- Quantas linhas o grid possui.
  4. grid-gap: 20px; - Como são os espaçamentos entre os elementos.
  5. grid-row-gap: ; - Como são os espaçamentos nas linhas.
  6. grid-column-gap: ;- Como são os espaçamentos nas colunas.
  7. grid-template-áreas:; - Delimita as áreas ;
  8. E 4 Propriedades de alinhamento para os containers:
  9. justify-content:; permite alinhar o prio grid no espaço fora do grid
  10. align-content:;permite alinhar o prio grid no espaço fora do grid
